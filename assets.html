Front-end + Firebase + Functions

Quick visual of how major files connect. Paths are repo-relative; edit here if you add new features.

UI Layer Data + Services Guides / Docs Automations

Entry & Routing

  • src/App.js → Theme, providers, router mount.
  • src/routes/Routes.js → Page map (dashboard, messaging, health, invites).
  • src/assets/theme/light.js → MUI theming.

Dashboard Surface

  • src/pages/PanelDashboard.js → Main dashboard shell.
  • src/components/Dashboard/ChildCard/ChildCardActions.js → Quick actions row.
  • src/components/Dashboard/ChildCard/SmsToggle.js → Per-child SMS logging toggle.
  • src/components/Dashboard/SmsEnablementTip.js → Nudge to enable SMS.

Messaging & SMS

  • docs/per-child-sms-implementation.md → Toggle behavior + UX rules.
  • docs/messaging-routing.md → Hashtag routing logic for inbound SMS/WhatsApp.
  • functions/ → Cloud Functions (e.g., updateChildSmsSettings).
  • src/pages/SettingsMessaging.js → User messaging settings page.

Data & Services

  • src/services/firebase.js → Firebase app, auth, Firestore, storage, functions.
  • src/contexts/ChildContext.js and src/contexts/RoleContext.js → Shared state.
  • src/services → API helpers (auth, messaging, storage).

Other Pages

  • src/pages/MedicalLog/MedicalLogPage.js → Medical entries.
  • src/pages/DailyLogPage.js + src/pages/TherapyNotesPage.js → Daily/therapy flows.
  • src/pages/MessagesPage.js → Inbox/history.
  • src/pages/ProfilePage.js and invite flows under src/pages/Invite*.

Testing & Ops

  • cypress/ + cypress.config.js → E2E setup.
  • src/tests/ → Jest unit/integration suites.
  • firebase.json + firestore.rules + firestore.indexes.json → Deployment + security.
  • scripts/*.js → Setup/migration helpers.
